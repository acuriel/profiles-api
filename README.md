# Profiles Test

## Requirements

For running this project you should have **Python >= 3.7** installed and added to your system path

## How to install dependencies

With pip:

> \> pip install -r requirements.txt

Or with [Poetry](https://python-poetry.org/)

> \> poetry

## How to execute tests

> \> python manage.py test

## How to execute the server

> \> python manage.py runserver

Django by default runs the server in the port **8000** but for avoiding collisions you can change it
> \> python manage.py runserver \<port>
